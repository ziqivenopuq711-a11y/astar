Astar Network Documentation

Astar Network is a Polkadot parachain (slot 2006) providing a multichain smart contract platform that supports both EVM-compatible (Solidity) and WASM-based (ink!) smart contracts. It bridges the Ethereum developer ecosystem with the security and interoperability of Polkadot, enabling cross-chain messaging via XCM and native dApp monetisation through Build2Earn staking.

Key Concepts

Parachain Architecture

Astar is secured by Polkadot validators as parachain 2006. Blocks are produced by Astar collators and finalized by the relay chain, inheriting Polkadot's shared security model without needing its own validator set.

Dual VM Execution

Astar runs two parallel virtual machines: the Frontier EVM layer (for Solidity/Vyper contracts using eth_* JSON-RPC) and the Substrate WASM runtime (for ink! contracts). Each has its own address space but can interoperate via precompiles.

dApp Staking (Build2Earn)

Developers register dApps in the staking registry. Token holders stake ASTR on registered dApps. At the end of each era, stakers earn a portion of ASTR inflation, and developers earn a developer reward share based on the total stake attracted.

Agile Coretime

With Polkadot 2.0, Astar benefits from Agile Coretime — a flexible execution core scheduling model. Instead of fixed parachain slots, Astar can purchase bulk coretime on the open market, enabling fine-grained resource allocation and potential on-demand block production.

Getting Started

  1. Register at dashboard.dwellir.com to get your API key.
  2. Choose your endpoint: HTTPS for request/response, WebSocket for subscriptions.
  3. Connect using @polkadot/api (Substrate) or ethers.js / viem (EVM).
  4. Deploy contracts on Shibuya testnet, graduate to Shiden canary, then Astar mainnet.
  5. Register your dApp in the Build2Earn staking registry to earn protocol rewards.
