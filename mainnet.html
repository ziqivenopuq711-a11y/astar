Astar Mainnet

Astar mainnet is a Polkadot parachain (slot 2006) secured by Polkadot's shared security. It supports both EVM and WASM smart contracts, features the Build2Earn dApp staking mechanism, and is fully compatible with Ethereum tooling via its Frontier EVM layer. The native token is ASTR, used for gas fees, staking, and governance.

Shiden — Canary Network

Shiden Network is Astar's canary network, deployed on Kusama as parachain 2007. It shares the same runtime codebase as Astar and serves as the final production test environment before mainnet upgrades. All major features (dApp staking, EVM, WASM) are validated on Shiden first.

Deployment Pipeline

The recommended workflow for new protocols is to progress through the network tiers:

  1. Shibuya testnet — Free test tokens (SBY), full feature parity, no economic risk.
  2. Shiden canary — Real SDN tokens, Kusama relay security, final integration testing.
  3. Astar mainnet — Production deployment with ASTR token, full Polkadot security.
