Developer Guide

Astar Network supports two developer tracks: EVM (Solidity/Vyper contracts using standard Ethereum tooling) and WASM (ink! contracts using Rust-based cargo-contract). Both tracks share the same underlying parachain infrastructure and can interoperate through Astar's XVM (Cross-Virtual Machine) precompiles.

EVM — ethers.js / viem

The Frontier EVM layer on Astar exposes a standard eth_* JSON-RPC surface. Any Ethereum library works out of the box. Replace the RPC URL with your Dwellir endpoint.

// ethers.js v6
import { JsonRpcProvider } from "ethers";

const provider = new JsonRpcProvider(
  "https://api-astar.n.dwellir.com/YOUR_API_KEY"
);

const blockNumber = await provider.getBlockNumber();
console.log("Astar block:", blockNumber);

// viem
import { createPublicClient, http } from "viem";
import { defineChain } from "viem";

const astar = defineChain({
  id: 592,
  name: "Astar",
  nativeCurrency: { name: "Astar", symbol: "ASTR", decimals: 18 },
  rpcUrls: {
    default: { http: ["https://api-astar.n.dwellir.com/YOUR_API_KEY"] },
  },
});

const client = createPublicClient({ chain: astar, transport: http() });
const block = await client.getBlockNumber();

Substrate — @polkadot/api

For Substrate-native queries (staking, governance, storage) use the Polkadot.js API over WebSocket. This exposes the full chain_*, state_*, and system_* RPC surface.

import { ApiPromise, WsProvider } from "@polkadot/api";

async function connect() {
  const provider = new WsProvider(
    "wss://api-astar.n.dwellir.com/YOUR_API_KEY"
  );
  const api = await ApiPromise.create({ provider });

  const header = await api.rpc.chain.getHeader();
  console.log("Best block:", header.number.toNumber());

  const [chain, nodeName, nodeVersion] = await Promise.all([
    api.rpc.system.chain(),
    api.rpc.system.name(),
    api.rpc.system.version(),
  ]);
  console.log(`Connected to ${chain} using ${nodeName} v${nodeVersion}`);
}

connect();

Smart Contract Deployment

EVM — Solidity (Hardhat / Foundry)

Configure hardhat.config.ts with Astar's network (Chain ID 592) and your Dwellir endpoint. Use Foundry's forge create for rapid deployments.

networks: {
  astar: {
    url: "https://api-astar.n.dwellir.com/KEY",
    chainId: 592,
    accounts: [PRIVATE_KEY]
  }
}

WASM — ink! (cargo-contract)

Install cargo-contract via Cargo. Build and deploy ink! contracts to Astar's Contracts pallet using the Contracts UI or CLI.

cargo install cargo-contract
cargo contract new my_contract
cargo contract build --release
cargo contract instantiate   --suri //Alice   --url wss://api-astar.n.dwellir.com/KEY
