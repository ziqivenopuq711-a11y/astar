Astar Network — Multichain Smart Contract Hub on Polkadot

Astar Network is a scalable smart contract platform on Polkadot supporting both EVM (Solidity) and WASM (ink!) execution environments. Native token ASTR powers the Build2Earn dApp staking ecosystem, enabling developers to earn while building production-grade decentralised applications. Agile Coretime and Asynchronous Backing provide ~6-second block times for a seamless developer experience.

Key Features

Dual VM — EVM + WASM

Deploy Solidity contracts via the Frontier EVM layer or ink! contracts on the WASM runtime. Full Ethereum JSON-RPC compatibility (eth_*, net_*, web3_*) alongside native Substrate APIs.

Build2Earn dApp Staking

Astar's flagship incentive mechanism lets developers register dApps and earn ASTR token rewards from protocol inflation. Users stake ASTR on their favourite dApps, creating a sustainable developer funding loop without relying solely on grants or VCs.

Polkadot 2.0 Ready

Astar has adopted Asynchronous Backing (~6-second block production) and Agile Coretime, the Polkadot 2.0 resource allocation model. This enables higher throughput and more flexible parachain scheduling without sacrificing security.

Managed RPC by Dwellir

Dwellir provides production-grade, load-balanced HTTPS and WebSocket endpoints for Astar, Shiden, and Shibuya. Endpoints are authenticated via API key and feature geo-distributed infrastructure for low-latency global access.

Canary Network — Shiden

Shiden Network is Astar's canary network on Kusama, sharing the same codebase but deployed earlier. It serves as the final testing ground before Astar mainnet upgrades. Native token SDN, EVM Chain ID 336.

Testnet — Shibuya

Shibuya is the public testnet for Astar Network, connected to the Tokyo relay chain. Developers can obtain free SBY test tokens from the faucet and test full EVM + WASM contract deployments before pushing to Shiden or Astar mainnet.
