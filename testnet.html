Shibuya Testnet

Shibuya is the public testnet for Astar Network, designed to replicate Astar mainnet conditions without economic risk. It runs on the Tokyo relay chain (Astar's dedicated test relay) and provides free SBY test tokens via a faucet. Shibuya supports full EVM and WASM contract deployment, dApp staking simulation, and XCM testing against the test relay chain.

Getting Test Tokens

SBY test tokens are available for free from the Astar Network faucet. Visit the official Astar portal faucet page and enter your Shibuya address (either EVM format 0x... or Substrate SS58 format). Tokens are credited within a few minutes and can be used for gas fees, contract deployment, and dApp staking tests.

Faucet: portal.astar.network → Faucet section → select Shibuya network

Connecting Your Wallet to Shibuya

MetaMask (EVM)

  1. Open MetaMask → Add Network → Add a network manually
  2. Network Name: Shibuya Testnet
  3. RPC URL: https://api-shibuya.n.dwellir.com/KEY
  4. Chain ID: 81
  5. Symbol: SBY

Polkadot.js (Substrate)

  1. Open polkadot.js.org/apps
  2. Top-left network selector → Custom endpoint
  3. Enter: wss://api-shibuya.n.dwellir.com/KEY
  4. Switch and confirm network connection

Testing Workflow

Follow this progression to safely validate your dApp before mainnet deployment:

  1. Shibuya testnet — Deploy contracts, test all user flows, validate dApp staking registration, simulate XCM transfers. No economic risk; use free SBY tokens.
  2. Shiden canary — Deploy with real SDN tokens, validate economic assumptions, final integration testing with Kusama relay chain security.
  3. Astar mainnet — Production launch with ASTR. Register in Build2Earn staking and begin attracting community stakers.
