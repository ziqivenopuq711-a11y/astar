RPC Methods Reference

Astar Network exposes two parallel JSON-RPC surfaces: the native Substrate API (chain, state, system, author namespaces) and the Ethereum-compatible Frontier EVM API (eth, net, web3 namespaces). Both are accessible via the same Dwellir endpoint. Use Substrate methods for parachain-native operations and Ethereum methods for EVM contract interactions.

Substrate RPC Methods

Method Description
chain_getHeaderReturns the header of a block by hash (or best head if omitted)
chain_getBlockReturns the full block (header + extrinsics) for a given hash
chain_getBlockHashReturns the block hash for a given block number
chain_getFinalizedHeadReturns the hash of the last finalised block
state_getStorageReads a storage value by key at an optional block hash
state_getKeysPagedPaginated enumeration of storage keys with prefix filter
state_getMetadataReturns the SCALE-encoded runtime metadata
state_getRuntimeVersionReturns current runtime spec and impl versions
system_healthReturns node health: peer count, syncing status, should have peers
system_chainReturns the human-readable chain name (e.g. "Astar")
author_submitExtrinsicSubmits a signed SCALE-encoded extrinsic to the transaction pool
payment_queryInfoEstimates fee for an extrinsic (partial fee + weight)

Ethereum-Compatible Methods (Frontier)

Method Description
eth_blockNumberReturns the current best block number as hex
eth_getBalanceReturns ASTR balance of an EVM address in wei
eth_callExecutes a read-only contract call without creating a transaction
eth_sendRawTransactionSubmits a signed RLP-encoded transaction to the mempool
eth_getTransactionReceiptReturns receipt for a mined transaction by hash
eth_getLogsReturns logs matching a filter (address, topics, block range)
eth_estimateGasEstimates gas usage for a transaction
eth_gasPriceReturns current base gas price in wei
net_versionReturns EVM chain ID as decimal string (e.g. "592" for Astar)
web3_clientVersionReturns node client identifier string

Request Examples

curl — eth_blockNumber

curl -X POST   -H "Content-Type: application/json"   -d '{
    "jsonrpc":"2.0",
    "method":"eth_blockNumber",
    "params":[],
    "id":1
  }'   https://api-astar.n.dwellir.com/YOUR_API_KEY

TypeScript — chain_getHeader

import { ApiPromise, WsProvider } from "@polkadot/api";

const api = await ApiPromise.create({
  provider: new WsProvider(
    "wss://api-astar.n.dwellir.com/YOUR_API_KEY"
  ),
});

const header = await api.rpc.chain.getHeader();
console.log("Block:", header.number.toNumber());
console.log("Hash:", header.hash.toHex());

Smoke Test Checklist

Run these checks after configuring your Dwellir endpoint to verify connectivity:

  1. Call system_health — verify isSyncing: false and peers > 0
  2. Call eth_blockNumber — confirm block number increments every ~6 seconds
  3. Call net_version — expect "592" for Astar mainnet
  4. Call chain_getFinalizedHead — verify finalized head is within 2 blocks of best head
  5. Subscribe via WebSocket to chain_subscribeNewHeads — confirm new headers arrive every ~6 seconds
